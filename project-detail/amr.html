Autonomous Mobile Robot

Project Overview

The Autonomous Mobile Robot (AMR) is a healthcare-focused robotic platform designed to support nurses and patients within hospital and aged care environments. Built with intelligent navigation and modular functionality, the robot aims to handle essential, repetitive tasks such as medication delivery, UV-C disinfection, and patient communication support.

This project focuses on reducing nurse workload and increasing operational efficiency through automation. It integrates a voice assistant for hands-free interaction, a medication payload compartment, and multiple modes including Patient Companion Mode and Sanitation Mode. The robot is being programmed with task scheduling, route optimization, and real-time obstacle avoidance using LIDAR and camera modules.

Key Features

  • Voice-assisted command interface
  • UV-C disinfection lights for sanitation tasks
  • Scheduled medication delivery system
  • Real-time navigation using SLAM algorithms
  • Modular design optimized for narrow hospital corridors

Technology Stack

  • Hardware: Custom AMR chassis, LIDAR, ultrasonic sensors, onboard computing unit
  • Software: ROS2, Python, OpenCV, speech recognition modules
  • Navigation: SLAM, path planning algorithms, dynamic obstacle avoidance
