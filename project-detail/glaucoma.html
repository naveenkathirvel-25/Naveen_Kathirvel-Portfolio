This project focuses on the early detection of glaucoma, a serious eye condition that can lead to blindness if untreated. By analyzing fundus images, the system leverages a combination of classical image processing and a ResNet-50 deep learning model to automate the identification of glaucoma-related features. The objective is to provide a fast, accessible, and scalable diagnostic tool suitable for clinical and telemedicine applications.

üß© Core Features & Methodology Highlights:

  • Image Preprocessing: Converts RGB fundus images into grayscale and applies contrast enhancement to highlight optic disc and cup boundaries.
  • Optic Disc & Cup Segmentation: Uses fuzzy edge detection and binarization to extract key regions.
  • CDR Calculation: Computes the Cup-to-Disc Ratio (CDR), a vital marker for glaucoma detection.
  • Deep Learning Classification: Employs a pretrained ResNet-50 architecture fine-tuned on a labeled dataset of glaucoma and non-glaucoma images.

üß™ Functionality & Performance:

  • Accurately classifies fundus images into glaucoma or normal using AI.
  • Reduces need for manual ophthalmologist screening in early diagnosis.
  • Trained on Kaggle dataset, achieving reliable accuracy with minimal preprocessing.
  • Utilizes morphological operations for refining input before feeding it into the network.

üè• Use Case Scenarios:

  • Rural Clinics: Enables remote eye screening using low-cost imaging and automatic classification.
  • Mobile Health Units: Supports early diagnosis through portable fundus cameras and embedded AI.
  • Hospital AI Systems: Can be integrated into electronic medical systems for patient triage and follow-up.
